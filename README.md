# hentai downloader

## Usage

1. package

	```bash
	./gradlew build
	```

2. run

   ```bash
   java -jar build/libs/nhentai-java-1.0-SNAPSHOT.jar
   ```

3. help

   ```bash
   shell:>help
   ```
   
4. download

   ```bash
   shell:>download --name xxx
   ```

5. convert

	```bash
	shell:>convert --path x --name y
	```
